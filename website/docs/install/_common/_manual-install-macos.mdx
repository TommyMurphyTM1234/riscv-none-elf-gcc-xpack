### Download

The macOS versions of **xPack GNU RISC-V Embedded GCC**
are packed as `.tar.gz` archives.
Download the latest version named like:

* <code>xpack-riscv-none-elf-gcc-{ customField('upstreamVersion') }-{ customField('xpackSubversion') }-darwin-x64.tar.gz</code>
* <code>xpack-riscv-none-elf-gcc-{ customField('upstreamVersion') }-{ customField('xpackSubversion') }-darwin-arm64.tar.gz</code>

:::note

In case you wonder where the suffix comes
from, it is exactly the Node.js `process.platform` and `process.arch`.

:::

### Unpack

To manually install the <b>xPack GNU RISC-V Embedded GCC</b>,
unpack the archive and move it to <code>~/Library/xPacks/riscv-none-elf-gcc</code> folder,
for example <code>/Users/ilg/Library/xPacks/riscv-none-elf-gcc/xpack-riscv-none-elf-gcc-{ customField('upstreamVersion') }-{ customField('xpackSubversion') }</code>:

<CodeBlock language="console"> {
`mkdir -p ~/Library/xPacks/riscv-none-elf-gcc
cd ~/Library/xPacks/riscv-none-elf-gcc

tar xvf ~/Downloads/xpack-riscv-none-elf-gcc-${ customField('upstreamVersion') }-${ customField('xpackSubversion') }-darwin-x64.tar.gz
